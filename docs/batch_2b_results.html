Rotated 45-degree beam (two sub-tests)

test_eb2d_rotated_beam.cpp NEW
PASS

45-deg cantilever in the xz-plane. Verifies the 2D coordinate transformation by (a) imposing a local transverse force and round-tripping through global, (b) imposing a global vertical force and checking the expected local axial/transverse coupling. This is the acceptance criterion specified in Codex Round 1 finding 4.

Sub-tests

3a Local transverse force PASS
Local +z tip force Flocal,z = −1000 on a 45-deg beam. The force is transformed to global via TT, the system solved, then tip displacement transformed back to local using T. The local tip displacement must equal the horizontal-reference solution to 1e−9 relative.
3b Global vertical force PASS
Global Fz = −1000 at the tip. With c = s = 1/√2, the local-force decomposition is Flocal,axial = +707.1 N and Flocal,trans = −707.1 N. Both global ux and uz at the tip are non-zero (axial/bending coupling) and match the independent local-axes solve to 1e−9.

Elimination + RHS correction. BeamLib stores the prescribed value in Node::prescribed; Node::totalDof returns this value for fixed DOFs. BeamModel::assemble reads totalDof when gathering element-local displacements, so the local internal force rl = kl dl automatically contains the Kfr ur RHS-correction term. No explicit Kfr matrix is formed. Reaction recovery then uses the full-system residual Rint − Fext; with no external load at the constrained DOFs, the reactions are pure internal residuals. See 01_euler_bernoulli_2d.tex §sec:prescribed.

Mass-matrix assembly (3 sub-tests)

test_eb2d_mass.cpp NEW
PASS

BeamModel::assembleMass() coverage. This is the Batch 8 / Batch 9 prerequisite: the dynamic-analysis and modal-analysis batches need a verified mass-assembly path before they can be implemented.

Sub-tests

5.1 Single horizontal element PASS
Beam along +x, λ = I, so Mg = Ml. Direct entry-by-entry check against the closed-form EB2D consistent mass: axial ρAL/6·[[2,1],[1,2]] + Hermite bending block ρAL/420·[156, 22L, 54, −13L; …]. Plus uTMu = ρAL for a rigid axial translation. Tolerance: relative 1e−12.
5.2 Vertical element (90°) PASS
Beam along +z, λ rotates global → local by 90°. After TTMlT: local axial mass maps to global uz diagonal (= 2ρAL/6), local bending translational mass maps to global ux diagonal (= 156·ρAL/420). The θy entries are unchanged because Tn(2,2)=1.
5.3 Two elements, fixed root PASS
Node 0 fully fixed; nodes 1 (interior) and 2 (tip) free. After DofMap reduction, Mff is 6×6. Node 1's diagonals receive contributions from both elements (B-end of elem 0 plus A-end of elem 1), so the axial diagonal there is 4ρAL/6 rather than 2ρAL/6. Symmetry check, accumulated entries, and matrix size all pass.
