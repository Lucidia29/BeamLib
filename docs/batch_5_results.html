What Batch 5 adds. 3D Timoshenko beam with the same DOF order as EB3D, composed of axial + St. Venant torsion + two independent Φ-corrected bending/shear planes. The xy plane uses EIz and κy·G·A (Φy = 12 EIz/(κy GA L²)); the xz plane uses EIy and κz·G·A (Φz = 12 EIy/(κz GA L²)). κy and κz stay independent throughout. Transformation reuses Batch 3's S·λ·S rotation adapter via BeamMath3D::buildTransformation3D. Mass adds two rotary-inertia deltas (ρ·Iy on θy, ρ·Iz on θz) on top of EB3D mass. ElementInternalForces was extended to carry the full 3D set {N, Vy, Vz, Tx, My, Mz}.

The torsion block (indices 3, 9) is identical to EB3D — Timoshenko introduces shear flexibility in the bending planes only, not in St. Venant torsion. This test confirms no regression to that inherited behavior.

Why torsion is unchanged from EB3D

St. Venant uniform torsion describes a beam in which:

  • cross-sections rotate rigidly about the beam axis,
  • no warping is included (Phase 1 scope),
  • shear strain in the cross-section is uniform after the κ correction is applied at the constitutive level.

There is no transverse-shear/bending-curvature coupling like Timoshenko's γ = u' − θ in the bending planes. The torsion block K = GIx/L · [[1,−1],[−1,1]] is identical between EB3D and Timoshenko 3D.

Verified by passing this test.

Why symmetric cross-section matters

"Slender" depends on the deflection direction.

For a rectangular section b × h:

  xz bending → "slenderness" = L/h

  xy bending → "slenderness" = L/b

If h ≠ b, the two planes have different effective slendernesses. To test "Timoshenko → EB in both planes" with a single tolerance, the cross-section must be symmetric (h = b) so both planes have L/(side) = 1000.

Used h = b = 0.001 m here.

Why two distinct ratios pin the constitutive wiring

With an asymmetric cross-section the two analytical ratios are different (1.04875 vs 1.02167). Any of the following bugs would shift one or both ratios:

  • swapping Iy ↔ Iz in either plane
  • swapping κy ↔ κz
  • using a single Φ for both planes instead of independent Φy/Φz
  • misassigning Φy to the xz plane (or vice versa)

Both ratios matching their analytical values to relative 1e−9 means the constitutive constants are correctly paired with their respective planes.

Why this is a key test for 3D Timoshenko. In 2D there is only one bending plane, so the constitutive constants have no ambiguity. In 3D the four numbers (Iy, Iz, κy, κz) can be mis-paired in subtle ways that would still pass a simple cantilever test if Iy=Iz or κy=κz. This test deliberately makes everything distinct.

Why no 3D locking

The locking-free property comes from each plane's Φ-corrected shape functions. Since the two planes are uncoupled in the local frame, locking-resistance carries over independently:

  • xz plane: uses Φz-corrected block with the same closed-form proof as 2D.
  • xy plane: uses Φy-corrected block independently.

No coupling means a single Φ swap (one plane vs. the other) would still leave both planes locking-free in isolation, but the deep-beam test catches that case via the ratio mismatch.

Two independent moderate-Φ spot checks confirm the (2 − Φ) sign in each plane separately. Asymmetric Iy ≠ Iz and κy ≠ κz mean a plane swap would shift the values by different amounts, making the check sensitive to per-plane bugs.

What this catches

The Φ-limit test is a high-resolution sanity check for the constitutive wiring:

  • Wrong Φ formula (e.g., 6 instead of 12) → entries off by factor of 2
  • Wrong plane mapping (e.g., Φy uses κz) → entries differ from EB3D limit
  • Sign error on (2 − Φ) → moderate-Φ spot check fails
  • Mixing Φy and Φz → cross-plane entries wrong

All 144 entries to rel 1e-7 is a strong constraint — any of the above bugs would fail multiple entries by orders of magnitude.

Mass-matrix strategy for Timoshenko 3D. Inherits the EB3D consistent mass (axial + Hermite transverse + ρIx torsional) and adds exactly two rotary-inertia 2×2 blocks, one per bending plane. Required by PROJECT_SPEC §3.4 for any Timoshenko 3D modal / dynamic analysis (Batches 8/9). Φ-dependent translational corrections (Friedman–Kosmatka closed-form mass) are deliberately deferred — the rotary-inertia delta is the dominant Timoshenko correction for slender-to-moderate beams.
